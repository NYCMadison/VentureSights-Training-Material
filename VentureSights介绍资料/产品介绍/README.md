## 🎯 快速了解产品（5分钟版）

### 我们是谁？

我们是**从产业分析引擎进化而来的科技投行智能体**。

**不是通用 AI，不是报告生成器，也不是「投行聊天机器人」**——而是 **场景可编排、交付可验收** 的科技投行专业工作体系（多角色协同 + Skills + 知识库），根在产业领域模型。

### 我们的演进历程

**第一阶段：产业分析引擎**
- 资源 **识别、评估、匹配** + 配套 **工具/智能体**；领域模型覆盖企业、产业链、投资机构、技术
- 目标客户：产业咨询公司、产业基金、并购/投融资服务机构
- 代表客户：中信咨询、中咨、Capgemini、民生银行、国泰君安创新投、北京产权交易所

**为什么要延伸到科技投行全链路（叙事要点）**
- 单点引擎再强，只占客户工作流一环；客户用 **归纳法** 评估价值需要足够长的链路与样本，否则难稳定回答「值不值」
- 团队常用 **演绎法** 强调「产业分析是核心」——两种推理都对，但要 **对齐评估框架**
- **延伸主因**：给客户更 **完整的工作流视角与可归纳评估的基础**；**加速器**：大模型与智能体框架降低扩展边际成本

**第二阶段：科技投行智能体（全场景 + 组织化交付）**
- 覆盖项目获取、尽调、交易结构、投决、市值与沟通、投后等链路；多 Subagent、Skill、知识库与 STOSS 编排
- **飞轮**：领域知识 **管理 + 推理** 在业务使用中沉淀增强，反哺下一单交付
- 吞吐与并行带来 **10～100 倍** 量级跃迁（口径见主介绍稿）

### 我们的核心突破

**架构层面的突破：**
- 从单体智能体到多Subagent专业化分工
- 从工具形态到服务输出多元化
- 从知识检索到多跳推理和因果分析

**能力层面的突破：**
- 产业理解的深度提升
- 知识推理的能力增强
- 风险识别的精度提高
- 投资建议的可操作性增强

**商业层面的突破：**
- 从单一的SaaS工具到多元服务输出
- 从产品销售到客户智能体外挂
- 从一次性交易到长周期合作
