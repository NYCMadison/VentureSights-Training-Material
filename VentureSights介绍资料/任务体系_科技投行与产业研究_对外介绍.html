一体化产品架构

对客户只呈现一个产品名、一套本地安装包；产业研究、尽调、日更、区域与交易材料等，均为智能体内的能力模块，在 Cursor / Claude Code 中由统一调度层按场景意图（及约定的 task_type）路由，共享预装知识库、质检门禁与虚拟项目组。

您接入的是什么

科技投行与产业研究智能体 — 在贵司本机或约定隔离环境安装的产业与交易研究执行层。 通过 Cursor / Claude Code 对话式下达任务，按场景生成不同研究包；无需为多个独立 Agent 分别采购与集成。

四层架构（由外到内）

① 本地化工作面 Cursor · Claude Code · 自然语言任务 · 本地落盘成稿（Markdown / Word 等）· 数据不出约定环境（以合同为准）
② 场景能力模块 产业 · 企业 · 日更（机会研究链 + V15）· VC 尽调 · 区域 · 交易材料 · PE 观察
③ 虚拟项目组 + 技能编排 多角色分工 + 可写进 SOW 的技能工序（Skills）
④ 预装领域知识库 + 数据能力 约 4000 篇企业分析报告 · 20 条产业链框架 · 全国市/区县产业集群数据接口 · 私有领域模型关键词（组）企业匹配 · 多领域专业知识库 · 工商/股东/行情 · 合规范围内公开信息获取

预装核心资产（知识库 + 数据接口，随安装交付）

企业分析报告库 · 约 4000 篇

覆盖过去 1 年内有融资事件的未上市企业；支撑标的初筛、可比对标、尽调底稿与企业章节快速起稿。可在本地检索、引用与增量更新（范围以合同为准）。

产业分析知识库 · 20 条产业链框架

内置 20 条产业链分析框架（含层级结构、节点定义、传导与机会分析口径），支撑产业深度报告、政策影响与技术/商业路径等注册框架分析。

产业集群分析数据 · 全国市·区县

预装覆盖全国城市、区县的产业集群分析数据接口（本地连接器）：集群指标、共现网络、综合评估与企业—集群映射等，支撑区域产业报告、招商研判与园区对标（授权与字段以合同为准）。

领域模型关键词（组）· 企业匹配

基于私有领域模型维护的产业/技术关键词与关键词组，对接企业匹配数据接口：按赛道语义扩展长名单、竞品与相似企业映射，与 4000 篇报告库、产业集群数据交叉引用。

为谁打包（典型客户）

咨询 · PE / VC · 战投

赛道扫描、标的初判、VC 尽调底稿、PE 观察清单、日更机会与 V15 战略面板。

并购 · 财务顾问 · ECM

行业章与 Teaser/IM 骨架、买方 Q&A、交易结构/税务/技术问题清单（非持牌意见）。

券商研究 · 园区招商

区域产业叙事、产业集群图谱与 Excel、政策—产业传导分析框架。

交付形态：研究向与交易材料向均为同一智能体内的模块，以本地化安装包一次交付；共享预装知识库、质检门禁与虚拟项目组。不提供对外 SaaS API 或浏览器控制台（特殊集成以合同约定为准）。

内置虚拟团队（Subagent）

领域知识库

随本地化安装预置的领域知识资产：约束产出结构、提升专业密度、支撑规模化交付。 分析任务默认采用规则检索 + 语义检索双路径，在本地加载相关知识条目。

四大预装底座（规模与口径）

  • 企业分析报告库（约 4000 篇）：覆盖过去 1 年内有融资事件的未上市企业，用于初筛、对标与尽调/企业章起稿。
  • 产业分析知识库（20 条产业链分析框架）：结构化产业链定义、节点与传导口径，支撑产业报告及三类注册框架（商业机会、技术路径、政策影响）。
  • 产业集群分析数据（全国市·区县）：覆盖全国城市、区县的产业集群指标、网络与评估数据接口，支撑区域产业与招商场景。
  • 领域模型关键词（组）企业匹配：基于私有领域模型的关键词/词组，对接企业匹配数据接口，扩展赛道长名单与可比企业池。

本地部署与数据能力

交付与使用均在本地完成：安装包内含智能体编排、预装知识库与可选数据连接器；分析任务与配置绑定，可走可重复流水线，减少口径漂移。

本地化安装（唯一对外交付形态）

  • 基于 Cursor、Claude Code（及约定 IDE 插件/规则包）安装与启用
  • 研究员以自然语言或场景模板下达任务，成品落盘至约定目录（Markdown / Word / HTML 等）
  • 按场景或 task_type 路由能力模块：产业/企业/区域、VC 尽调、PE 观察、日更+V15、交易材料等
  • 环境隔离、目录权限与升级策略以安装手册及合同为准（非对外 SaaS API）

结构化数据输入

  • 企业名单、统一社会信用代码、关注指标序列
  • 标签与元数据、客户内部主数据 ID
  • 可比公司列表、筛选条件与对比维度
  • 与贵司数据平台、投研系统或 CRM 直连（数据进、报告出）

公开信息获取

  • 新闻、政策文件、监管与披露网页（合规与站点政策范围内）
  • 与知识库条目、框架字段映射，标注时效
  • 补足「最新发生了什么」，可回答来源与时点追问

企业主体事实

  • 工商登记基础字段（名称、信用代码、法人、注册资本等）
  • 股东、出资与控制关系线索（以授权数据源为准）
  • 相似企业与竞品推荐（产业关键词 + 双路合并）
