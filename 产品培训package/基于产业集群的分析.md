# 基于产业集群的分析

和产业链分析不同，产业集群分析是bottom-up的分析方法；根据给定企业范围（城市，园区，产业链），以及 VentureSights系统对这些企业的画像，可以构建产业集群图谱（企业基于技术和业务相似程度所形成的网络）；并且根据企业在产业集群图谱中的拓扑结构来做集群的划分；有相似网络拓扑结构的企业更加倾向于属于同一个产业集群（或者分类）；

将企业绘制到产业集群图谱中所带来的好处非常明显：1）企业在产业集群图谱中所提现的拓扑结构可以被理解为是高维度数据在2D图中的一个投影，企业之间的“边”提现的是彼此的业务/技术相似程度；这种高维度数据是VentureSights对企业做画像和向量化的结果，反映了企业在复杂的产业上下游环境中的相对位置；产业集群图谱所构成的复杂网络以及相关量化的指标可以系统化的分析企业之间竞争与合作的关系；而这些是产业分析急迫需要解决的问题；2）以这些高维度数据为基础，所构建的分析框架和模型，可以借助算力来形成高效的产业分析知识推理；
